## 4. シンボルマーク構成

シンボルマークの詳細仕様（寸法・全120セルのカラーコード・モザイクパターン）は
**[icon-specification.md](icon-specification.md)** を参照。

| 項目 | 値 |
|------|------|
| **構成要素** | 3列 × 4段の角丸矩形（合計 12 個） |
| **内部構造** | 各矩形は 5×2 セルで塗り分け（全120セル） |
| **矩形寸法** | 100 × 40 px（5 : 2） |
| **角丸半径** | 10 px（矩形高さの 25%） |
| **矩形間隔** | 10 px（列間・行間とも） |
| **シンボル全体** | 320 × 190 px |
| **配置** | ロゴタイプとシンボルの間隔はロゴタイプ高さの約 1 倍 |
